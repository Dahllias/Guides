# Guides

The source of SkyKings' guides.

## Contributing

If you want to contribute to the guides, you can [add a new one](https://github.com/SkyKings-Guild/Guides/new/main/guides), 
or edit an existing one by visiting the guide and clicking the "Edit" button.
- On GitHub, the edit button will be in the top right of the file.
- On [SkyKings](https://skykings.net), the edit button will be in the bottom right.

Guides are written in Markdown, you can find a syntax overview at https://www.markdownguide.org/cheat-sheet/.

You can find an example guide in [guides/example.md](guides/example.md).
